# JavaScript Array Methods Cheat Sheet

This repository contains simple examples of commonly used JavaScript array methods.

## Methods Covered
- map()
- filter()
- reduce()
- find()
- forEach()

Each file contains clear examples with comments.

## Why Learn Array Methods?
Array methods help write cleaner and more readable JavaScript code.

## Example

const numbers = [1,2,3,4,5];

const doubled = numbers.map(num => num * 2);

console.log(doubled);
// Output: [2,4,6,8,10]
